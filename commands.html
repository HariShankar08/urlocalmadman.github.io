!mute <@user or user id> <time> <reason(optional)>
Mutes the user for the specified amount of time(hours = h, minutes = m seconds = s).
If not specified, time defaults to 30 minutes.
Example: !mute @retard 30m spamming. This mutes the user @retard for 30 minutes for spamming(lives up to his name).



!unmute <@user or user id>
Unmutes a previously muted user.
Example: !unmute @retard. The user retard will have the muted role removed(if he had it previously).
